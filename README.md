# TechForge ERP - Saiyan Edition

Dự án ERP quản lý nhân sự và dự án.

## Cấu trúc dự án
- `/src`: Mã nguồn Backend (Java).
- `/web`: Mã nguồn Frontend (HTML, CSS, JS).

## Cách chạy dự án
1. **Frontend:** Mở file `web/index.html` bằng trình duyệt.
2. **Backend:** (Đang cập nhật) Chạy file `src/Main.java`.
